# Practica 7 - Alan Barajas

## Personaje
- Nombre del personaje: Leonix

## Controles
- W: adelante.
- S: atras.
- A: izquierda.
- D: derecha.

- Estos son vehiculos que aparecen desde la izquierda o derecha y se mueven en horizontal siguiendo su linea
